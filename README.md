## Usage 🔭

Read more about the [Design](https://github.com/intellibus/biospheres/blob/main/DESIGN.md) behind `biospheres` here.

`.solarsystem.json`

```json
{
  "lang": "biospheres/javascript",
  "iac": "biospheres/terraform",
  "cloud": ["biospheres/aws", "biospheres/gcp"],
}
```
